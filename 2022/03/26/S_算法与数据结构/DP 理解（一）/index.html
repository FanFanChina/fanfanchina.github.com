DP 理解（一）

发表于2022-03-26|更新于2022-05-14|S_算法与数据结构
|阅读量:

何为DP?

01.用前面的数据去更新后面的数据，更新时根据属性选择更新方式——max、min、sum

02.动态规划即我们的规划方式是动态的，当下的觉决策来源于之前数据，之前的数据一变，决策结果也会随之改变

03.状态转移方程其实就是更新时的更新方式，思路一般来源于题目中蕴含的操作

例子

01背包问题

这道动态规划的入门题目就蕴含了上述的3点

属性：max

操作：对第i个物品选或不选

完全背包

属性：max

操作：选择k个第 i 个物品

最长上升子序列

属性：max

操作：用第i个数和前面所有的前缀的最后一个数比较，如果大于就更新f[i] = f[前缀长度] + 1，但这只是其中一个前缀对应的f[i]，为了求最大值，f[i] = max(f[i], f[前缀长度] + 1);

挖地雷

属性：max

操作：前缀路径 + a[i] 和 f[i] 比较

编辑距离

属性：min

操作：题目已给出删除、插入、修改 + 不变

最长公共子序列

属性：max

操作：有新的公有元素 + 1，没有则继承之前的

括号序列

属性：min

操作：以右括号为基准，添加左括号，遇见左括号继承，右括号添加k个左括号
